## [+] Installation & Usage :atom_symbol:
```
apt update && upgrade -y 
pkg install git python mpv figlet -y
pip install lolcat
git clone https://github.com/tarunstarkjr/Termux-by-Stark
cd Termux-by-Stark
chmod +x *
sh install.sh
exit
```

